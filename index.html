Пишу компанды по шиту

Основные компанды

  • git clone path to repository
  • git status - статус репозитория
  • git add файл - трекает файл (гит контролит файл)
  • git commit -am "added new file index.html"
  • git push - отправить закомичченный код в репозиторий на сервер git
  • git pull - забрать с репозитория github код
